# Бот кибербезопасности для MAX
Чат-бот для мессенджера MAX, который помогает:

1. Проверять URL на фишинг (через чёрный список);
2. Проходить адаптивные тесты по кибербезопасности (по возрастным группам).
## Локальный запуск
  Требования:
  
- Docker;

- Python 3.11+ (опционально, для разработки)
## Сборка контейнера
    docker build -t max_bot .

##  Запуск контейнера
    docker run --env-file .env max_bot
## Важно

Бот не может писать первым — пользователь должен начать диалог (например, отправить /start или любое сообщение).

    Бот реализован с использованием официального API MAX https://dev.max.ru/docs
